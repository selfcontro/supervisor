# FlowDiagram 设计文档

## 1. 概述
FlowDiagram 是 Agent Team Visualizer 的核心可视化组件，负责以流程图形式展示 Agent 层级关系和任务流向。

## 2. 技术选型
- **库**: React Flow (基于 xhtml2svg)
- **版本**: Next.js 14 兼容版本
- **布局算法**: Dagre (用于自动布局)

## 3. 节点设计

### 3.1 节点类型

#### Planner Node
```
┌─────────────────────┐
│     🧠 Planner      │
│  ─────────────────  │
│  状态: 空闲         │
│  当前任务: 无        │
└─────────────────────┘
```

#### Executor Node
```
┌─────────────────────┐
│     ⚡ Executor     │
│  ─────────────────  │
│  状态: 工作中       │
│  当前任务: 排序算法  │
└─────────────────────┘
```

#### Reviewer Node
```
┌─────────────────────┐
│     👀 Reviewer     │
│  ─────────────────  │
│  状态: 空闲         │
│  当前任务: 无        │
└─────────────────────┘
```

### 3.2 节点状态颜色
| 状态 | 边框颜色 | 背景色 | 图标 |
|------|----------|--------|------|
| 空闲 (idle) | #22c55e (绿色) | #f0fdf4 | 灰色 |
| 工作中 (working) | #eab308 (黄色) | #fefce8 | 动画脉冲 |
| 完成 (completed) | #3b82f6 (蓝色) | #eff6ff | 绿色勾 |
| 错误 (error) | #ef4444 (红色) | #fef2f2 | 红色叉 |

### 3.3 节点尺寸
- 宽度: 200px
- 高度: 120px
- 圆角: 12px
- 阴影: 0 4px 6px rgba(0,0,0,0.1)

## 4. 边/连线设计

### 4.1 连线类型
- **Planner → Executor**: 实线，蓝色 (#3b82f6)
- **Executor → Reviewer**: 实线，蓝色 (#3b82f6)
- **Planner → Reviewer**: 虚线，灰色 (#9ca3af)

### 4.2 任务流动画
当任务在 Agent 间流转时，连线上显示移动的圆点动画：
- 动画时长: 2秒
- 动画类型: 线性移动
- 圆点大小: 8px

### 4.3 活动指示
- 当前执行的连线: 箭头闪烁动画
- 完成任务连线: 绿色渐变

## 5. 布局

### 5.1 默认布局 (Dagre)
```
     ┌────────────┐
     │  Planner   │
     └─────┬──────┘
           │
     ┌─────▼──────┐
     │  Executor  │
     └─────┬──────┘
           │
     ┌─────▼──────┐
     │  Reviewer  │
     └────────────┘
```

### 5.2 布局参数
```javascript
const dagreOptions = {
  rankdir: 'TB',      // 从上到下
  nodesep: 80,        // 节点间距
  ranksep: 100,       // 层级间距
  marginx: 50,       // 水平边距
  marginy: 50        // 垂直边距
};
```

## 6. 交互设计

### 6.1 节点交互
| 操作 | 行为 |
|------|------|
| 单击 | 选中节点，打开详情面板 |
| 双击 | 聚焦节点，缩放适配 |
| 拖拽 | 移动节点位置（禁止） |

### 6.2 画布交互
| 操作 | 行为 |
|------|------|
| 滚轮 | 缩放 (0.5x - 2x) |
| 拖拽空白处 | 平移画布 |
| 双击空白处 | 重置视图 |

### 6.3 快捷键
| 快捷键 | 行为 |
|--------|------|
| R | 重置视图 |
| F | 聚焦选中节点 |
| +/- | 放大/缩小 |

## 7. 性能要求
- 节点数量 <= 50 时保持 60fps
- 动画使用 CSS transform（GPU 加速）
- 大列表使用虚拟滚动

## 8. 状态管理
- 使用 Zustand 管理流程图状态
- 状态持久化到 localStorage（节点位置等）

## 9. 辅助功能
- 键盘导航支持 (Tab 切换节点)
- 高对比度模式支持
- 暗色模式适配
